# 🚀 TechForAll-Site

Plataforma de mentoria voluntária em tecnologia desenvolvida com HTML, CSS e JavaScript.

---

## 🌐 Sobre o Projeto

O **TechForAll** é um site criado como projeto acadêmico da disciplina de **Programação Front-End**, com o objetivo de conectar mentores da área de tecnologia a pessoas que desejam aprender programação, informática e novas tecnologias de forma gratuita.

O projeto busca incentivar o aprendizado acessível, promovendo a troca de conhecimento entre profissionais e iniciantes.

Desenvolver um site interativo e responsivo que permita:

* Apresentar a proposta de mentoria
* Exibir cursos disponíveis
* Conectar mentores e alunos
* Permitir inscrição através de formulários
