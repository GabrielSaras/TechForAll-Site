🚀 TechForAll-Site

Plataforma de mentoria voluntária em tecnologia desenvolvida com HTML, CSS e JavaScript.

O objetivo do projeto é conectar mentores da área de tecnologia com pessoas que desejam aprender programação, informática e novas tecnologias de forma gratuita.

🌐 Sobre o Projeto:

O TechForAll é um site criado como projeto acadêmico da disciplina de Programação Front-End (Lecionado por: Porf° Jose Flores - ADS - A 3° Perido na Unicesumar - 2026).

A plataforma busca incentivar o aprendizado em tecnologia através de mentoria voluntária, permitindo que profissionais compartilhem conhecimento com iniciantes.

🛠️ Tecnologias Utilizadas:

HTML → Estrutura do site

CSS → Estilização e layout

JavaScript → Interatividade e funcionalidades
